### About
I’m a frontend developer focused on building fast, clean websites for brands and vendors. This portfolio highlights real projects I’ve shipped: restaurant sites, event ticket systems, solar company sites, and mini e-commerce stores with WhatsApp checkout.

No bloat. Just performance + clean UX.

### Features
- **Project Showcase**: Detailed modals with screenshots, tech stack, and results for each project
- **Testimonials**: Real client feedback with WhatsApp/email screenshots 
- **Responsive Design**: Mobile-first layout, tested on all screen sizes
- **Smooth Animations**: Custom loader, scroll reveals, testimonial slider
- **Contact Form**: Direct email integration for inquiries
- **Dark Theme**: Modern dark UI optimized for readability
- **Fast Loading**: Optimized images + vanilla JS, no heavy libraries

### Tech Stack
`HTML5` `CSS3` `Vanilla JavaScript` `Responsive Design` `Vercel/Netlify`
